GitHub Pagesでの公開手順

Step 1: GitHubアカウントとリポジトリの作成

GitHubのアカウントを作成し、新しいリポジトリを作成します。リポジトリ名は「username.github.io」とするとメインサイトになります。

Step 2: リポジトリにファイルをアップロード

作成したHTMLファイル、CSSファイル、JavaScriptファイル、および画像をリポジトリにアップロードします。

Step 3: GitHub Pagesの設定

リポジトリの「Settings」タブをクリックし、左側のメニューから「Pages」を選択します。「Source」セクションでブランチ（通常は「main」）を選択し、保存します。

Step 4: サイトの公開

設定が完了すると、「Your site is published at https://username.github.io/」というメッセージが表示され、サイトが公開されます。

Step 5: カスタムドメインの設定（オプション）

独自ドメインを使用したい場合は、「Custom domain」セクションにドメイン名を入力し、DNSプロバイダーでCNAMEレコードを設定します。

詳細な手順については、GitHub Docsの公式ページをご参照ください。
